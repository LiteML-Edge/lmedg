Table I DOI audit package. This folder contains the final matrix, the evidence log, the source index, and the combined spreadsheet used to support Table I. A check mark is assigned only when the corresponding element is explicitly reported in the reviewed paper. Blank cells are descriptive only and indicate that the element was not explicitly identified in the focused review used to populate Table I. The package preserves DOI/source mapping and criterion-level evidence for review of the representative works. It does not rely on bundled copies of the cited articles.
